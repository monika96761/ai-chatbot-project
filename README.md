# AI Chatbot using Groq API

This project contains multiple AI chatbot implementations using Python.

## Features
- Streamlit web chatbot
- Terminal chatbot
- Multimodal image chatbot
- Groq API integration

## Technologies
- Python
- Streamlit
- Groq API
- HuggingFace API

##API Key Setup

To run this project, you must provide a Groq API key.
 Add the API key directly in ch.py
Open the file ch.py and replace:
api_key = "YOUR_API_KEY"
with your actual API key.
Example:
api_key = "gsk_your_api_key_here"

## Install Dependencies

pip install -r requirements.txt

## Run Web Chatbot

streamlit run bot.py
